NAME

dbish - Interactive command shell for the Perl DBI

SYNOPSIS

  dbish <options> dsn [user [password]]

DESCRIPTION

This tool is a command wrapper for the DBI::Shell perl module. See the DBI::Shell(3) manpage for full details.

SEE ALSO

the DBI::Shell(3) manpage, DBI(3)
